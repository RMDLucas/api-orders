# API Orders

API desenvolvida em **Node.js** com **Express** e **MongoDB** para gerenciamento de pedidos.

Este projeto implementa uma API REST que permite **criar, consultar, listar, atualizar e deletar pedidos**, realizando também o **mapeamento dos campos do JSON de entrada** antes de salvar no banco de dados.

---

# Tecnologias utilizadas

- Node.js
- Express
- MongoDB
- Mongoose
- Nodemon
